# Contributing to the Solas Project
We are a small team of developers who currently want to keep our code _closed_ for pull requests so we have full control over it.
However, we want the community to **discuss new features and report bugs**, so we have open **discussion and issue pages**.
You are free to discuss anything as long as it doesn't violate the [Code of Conduct](https://github.com/NevskyV/Solas/blob/master/CODE_OF_CONDUCT.md).
